# SSMS - Shagun School Management System

Advanced, Multi-Tenant School ERP Ecosystem. Developed by Shagun Systems.

## Features

- **Multi-Tenant Architecture**: Isolate data across different schools.
- **Master Admin Control**: Global registry to manage schools, payments, and subscription plans.
- **Academic Suite**: Manage students, staff, exams, and attendance.
- **Financial Module**: Advanced fee collection and payroll systems.

## Getting Started

1. Install dependencies: `npm install`
2. Run development server: `npm run dev`
